# Petals n Posies 🌸
A full stack e-commerce platform for a local florist shop. 
Customers can browse flower arrangements, select variants, 
pick a delivery date, and checkout via Stripe. 
Admins can manage inventory and view orders.

## Live Demo
https://petals-n-posies.vercel.app

## Tech Stack
**Frontend:** React, Redux Toolkit, Tailwind CSS, React Router
**Backend:** Spring Boot, Spring Security, JWT, PostgreSQL
**Payment:** Stripe Checkout + Webhooks
**Deployment:** Vercel (frontend), Railway (backend)

## Features
- Browse flower arrangements with variant pricing
- Cart management with delivery date selection
- Stripe payment integration with webhook confirmation
- Admin dashboard for inventory management
- JWT authentication for admin access
- Order management with daily delivery view
- Responsive design
- Email notifications

## Known Issues
- Login does not redirect after authentication
- Logout not yet implemented
- Tax calculation not enabled on Stripe checkout
- Admin order page not displaying orders after payment
- Mobile header needs responsive fixes
